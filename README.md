# Global Super Store Data Analysis Project (Power BI)

## 👥 Team Collaboration
* **Team size:** 2 members
* **Role:** Co-developer
* **Objective:** Collaborated to break down business requirements, process raw datasets, and co-develop an interactive 4-page dashboard to optimize sales performance and customer segmentation.

---

## 🛠️ Tech Stack & Skills Used
* **Tool:** Microsoft Power BI
* **Data Processing:** Power Query (Data cleaning, transformation, and multi-market data validation).
* **Analytics:** Advanced DAX (Data Analysis Expressions) for customer tiering, growth rate calculations, and dynamic metrics tracking.
* **Data Modeling:** Star Schema / Relationship building for seamless cross-filtering across 4 pages.
