# Global Super Store Data Analysis Project (Power BI)

## 👥 Team Collaboration
* **Team size:** 2 members
* **Role:** Co-developer
* **Objective:** Collaborated to break down business requirements and co-develop an interactive 4-page dashboard to solve corporate sales and customer retention challenges.

* **Tool:** Microsoft Power BI
* **Data Processing:** Power Query (Data cleaning & transformation)
* **Analytics:** Advanced DAX (Data Analysis Expressions) for customer tiering and metrics tracking.
* **Data Modeling:** Star Schema / Snowflake Schema.
